## Proje Kurulumu ve Çalıştırılması
1. Projeyi klonlayın veya indirin.
2. Gerekli Flutter bağımlılıklarını yükleyin:
   ```
   flutter pub get
   ```
3. Uygulamayı başlatın:
   ```
   flutter run
   ```
